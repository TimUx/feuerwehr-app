# Screenshots

This directory contains screenshots for the README documentation.

## Screenshot Guidelines

- **Resolution**: 1080x1920px (Full HD Mobile 9:16 aspect ratio)
- **Format**: PNG
- Include both light and dark mode where applicable
- Show the actual UI with sample data
- Ensure sensitive information is removed or anonymized
- Text, symbols, and UI elements must be clearly visible and properly rendered
